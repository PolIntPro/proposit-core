# 0.6.0

## Hierarchical Checksums

Every hierarchical entity (expression, premise, argument) now carries three
checksum fields instead of one:

- `checksum` — meta checksum derived from entity data only (driven by
  `checksumConfig`)
- `descendantChecksum` — derived from children's `combinedChecksum` values;
  `null` for leaves / empty collections
- `combinedChecksum` — equals `checksum` when no descendants, otherwise
  `computeHash(checksum + descendantChecksum)`

### New features

- **Bottom-up dirty propagation** — mutations mark affected entities and all
  ancestors as checksum-dirty. Recomputation is lazy, triggered by
  `flushChecksums()` or by accessing any checksum property.
- **Per-collection checksums** — `getCollectionChecksum("premises")`,
  `getCollectionChecksum("variables")`, `getCollectionChecksum("expressions")`
  return checksums scoped to individual descendant collections.
- **Checksum verification on load** — `fromSnapshot` and `fromData` accept
  `checksumVerification?: "ignore" | "strict"`. Strict mode throws on mismatch
  between stored and computed checksums.
- **`THierarchicalChecksummable<TCollectionName>`** interface replaces
  `TChecksummable`, exposing `checksum()`, `descendantChecksum()`,
  `combinedChecksum()`, `getCollectionChecksum()`, and `flushChecksums()`.

### Breaking changes

- `TChecksummable` interface removed — use `THierarchicalChecksummable`
- `ArgumentEngine.checksum()` and `PremiseEngine.checksum()` now return the
  **meta** checksum (entity data only). Previously they returned a composite
  hash mixing entity and descendant data. Use `combinedChecksum()` for the
  old "did anything change?" behavior.
- Expression, premise, and argument entities now have required
  `descendantChecksum` and `combinedChecksum` fields.
- `TOptionalChecksum<T>` broadened to handle all three checksum fields.
- `TOptionalHierarchicalChecksum<T>` added for hierarchical entities.
- Role state now contributes to the argument's meta checksum instead of being
  a separate entry in the old composite hash.
