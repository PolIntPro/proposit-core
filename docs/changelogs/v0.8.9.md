# Changelog

## Auto-Normalize: Formula Collapse & Gated Operator Collapse

`9b4a58c..690a8ef`

### Breaking Changes

- `DEFAULT_GRAMMAR_CONFIG.autoNormalize` changed from `false` to `true`. Callers relying on the default config now get auto-normalization (formula buffer insertion + operator/formula collapse). Use `PERMISSIVE_GRAMMAR_CONFIG` or explicit `{ autoNormalize: false }` to opt out.
- Operator collapse (0/1 children after removal) is now gated on `autoNormalize`. With `autoNormalize: false`, operators with 0 or 1 children survive removal.
- Formulas wrapping non-binary-operator subtrees are now collapsed when `autoNormalize` is `true`. Trees that previously retained formulas around variables or `not` chains will have those formulas removed.

### New Features

- **Formula collapse rule:** `collapseIfNeeded` extended to collapse formulas whose bounded subtree (stopping at the next nested formula) has no `and`/`or` operator.
- **`hasBinaryOperatorInBoundedSubtree`:** New private helper on `ExpressionManager` for the bounded subtree check.
- **Recursion after operator promotion:** `collapseIfNeeded` now recurses to the grandparent after operator promotion, enabling cascade through parent formulas.
- **`removeAndPromote` cascade:** `removeAndPromote` now calls `collapseIfNeeded` on the target's parent after promotion.
- **`ExpressionManager.normalize()`:** Public on-demand normalization sweep (collapse operators, collapse formulas, insert formula buffers). Works regardless of `autoNormalize`.
- **`PremiseEngine.normalizeExpressions()`:** Normalizes one premise's expression tree, returns changeset.
- **`ArgumentEngine.normalizeAllExpressions()`:** Normalizes all premises, returns combined changeset.
- **Post-load normalization:** `fromSnapshot` and `fromData` run a normalization sweep after loading when `autoNormalize` is `true`.
- **`simulatePostPromotionCollapse`:** New private method extending removal safety simulation for formula collapse cascades.
- **`promoteChild`:** Extracted shared promotion logic from `collapseIfNeeded` into a reusable private helper.

### Internal Changes

- `simulateCollapseChain` extended for formula 1-child collapse and post-operator-promotion recursion.
- `assertRemovalSafe` updated to simulate post-promotion cascade via `simulatePostPromotionCollapse`.
- `TGrammarConfig` JSDoc updated to document formula collapse rule and `autoNormalize` scope.
- `TExpressionMutations` interface updated with `normalizeExpressions()` signature.
- CLAUDE.md updated with formula collapse and operator collapse gating design rules.
